# 장기 AI 웹 게임 개발 가이드

이 문서는 Flask 프레임워크를 사용하여 개발된 웹 기반 장기 게임의 코드 구조, 기능 및 사용 방법을 설명합니다.

## 1. 프로젝트 개요

이 프로젝트는 웹 브라우저에서 사용자가 장기를 플레이하고, AI(Fairy Stockfish 엔진)와 대결할 수 있는 간단한 장기 게임을 구현하는 것을 목표로 합니다. 장기 규칙에 맞는 기물 이동 로직과 AI 연동이 핵심 기능입니다.

## 2. 기술 스택

*   **백엔드**: Python 3.x, Flask
*   **프론트엔드**: HTML5, CSS3, JavaScript (ES6+)
*   **AI 엔진**: Fairy Stockfish (장기 모드 활성화)
*   **엔진 연동**: Python `subprocess` 모듈

## 4. 구현된 주요 기능

### 4.1. 장기판 및 기물 렌더링

*   HTML, CSS, JavaScript를 사용하여 10x9 장기판을 웹에 표시합니다.
*   `static/img` 폴더의 SVG 이미지 파일을 사용하여 다양한 스타일의 장기말을 렌더링합니다.
*   **초기 배치**: 표준 장기 규칙에 맞는 기물 초기 배치를 구현했습니다.

### 4.2. 기물 이동 로직 (프론트엔드 - `static/js/game.js`)

각 기물의 고유한 이동 규칙이 구현되어 있습니다.

*   **병(P) / 졸(p)**:
    *   상하좌우 한 칸 이동.
    *   상대방 궁성 내에서는 대각선 한 칸 이동 가능.
*   **마(N) / 마(n)**:
    *   상하좌우 한 칸 이동 후 대각선 한 칸 이동.
    *   **멱 규칙 적용**: 중간 경유지에 기물이 있으면 이동 불가.
*   **포(C) / 포(c)**:
    *   상하좌우 직선 이동.
    *   반드시 **하나의 기물(산)**을 뛰어넘어야 함.
    *   **포끼리는 서로 넘을 수 없음**.
    *   궁성 내에서는 대각선으로 기물을 뛰어넘을 수 있음.
*   **상(B) / 상(b)**:
    *   상하좌우 한 칸 이동 후 대각선 두 칸 이동.
    *   **멱 규칙 적용**: 첫 번째 상하좌우 경유지와 두 번째 대각선 경유지 모두 비어있어야 이동 가능.
*   **차(R) / 차(r)**:
    *   상하좌우 직선으로 원하는 만큼 이동.
    *   중간에 기물이 있으면 이동 불가 (상대 기물은 잡고 멈춤, 아군 기물은 그 앞에서 멈춤).
*   **궁(K) / 궁(k) 및 사(A) / 사(a)**:
    *   궁성 내에서 상하좌우 및 대각선으로 한 칸 이동.
    *   궁성 밖으로는 이동 불가.

### 4.3. 사용자 인터페이스 (UI)

*   **기물 선택 표시**: 기물을 클릭하면 기물 아래에 파란색 원이 표시됩니다.
*   **유효 이동 경로 표시**: 선택된 기물이 이동할 수 있는 모든 유효한 칸에 녹색 사각형이 표시됩니다.
*   **게임 상태 표시**: 현재 턴인 플레이어, 게임 상태 메시지, 현재 FEN 문자열이 UI에 표시됩니다.

### 4.4. AI 연동 (백엔드 - `app.py`)

*   **Fairy Stockfish 엔진 사용**: `stockfish.exe`를 `subprocess` 모듈을 통해 실행하고 통신합니다.
*   **UCI 프로토콜**: 엔진과 UCI(Universal Chess Interface) 프로토콜을 사용하여 통신합니다.
*   **장기 모드 활성화**: `setoption name UCI_Variant value janggi` 명령을 통해 엔진의 장기 모드를 확실하게 활성화합니다.
*   **최적의 수 계산**: 현재 장기판 상태(FEN)를 엔진에 전달하고, 엔진이 계산한 최적의 수(UCI 형식)를 받아옵니다.
*   **AI 평가 표시**: 엔진이 제공하는 `score cp` (센티폰 점수) 또는 `score mate` (체크메이트까지 남은 수) 정보를 파싱하여 프론트엔드로 전달하고 UI에 표시합니다.

### 4.5. 효과음

*   기물이 이동할 때 (사용자 및 AI 모두) `static/audio/move.wav` 파일의 효과음이 재생됩니다.

## 5. 사용 방법

### 5.1. 사전 준비

1.  **Python 설치**: Python 3.x 버전이 설치되어 있어야 합니다.
2.  **Flask 설치**: 명령 프롬프트(또는 터미널)에서 다음 명령어를 실행하여 Flask를 설치합니다.
    ```bash
    pip install Flask
    ```
3.  **Fairy Stockfish 엔진**: `stockfish.exe` 파일이 프로젝트의 루트 디렉토리(`ai_janggi/`)에 있어야 합니다. (Windows 기준)
4.  **효과음 파일**: `move.wav` 파일이 `ai_janggi/static/audio/` 디렉토리에 있어야 합니다. (다른 이름이나 형식이라면 `static/js/game.js` 파일의 `moveSound` 경로를 수정해야 합니다.)

### 5.2. 애플리케이션 실행

1.  명령 프롬프트(또는 터미널)를 열고 프로젝트 루트 디렉토리(`C:\Users\kjlkm\my_app\ai_janggi\`)로 이동합니다.
2.  다음 명령어를 실행하여 Flask 서버를 시작합니다.
    ```bash
    python app.py
    ```
3.  서버가 성공적으로 시작되면, 다음과 유사한 메시지가 표시됩니다.
    ```
     * Serving Flask app 'app'
     * Debug mode: on
     * Running on http://0.0.0.0:5000
    ```
4.  웹 브라우저를 열고 `http://127.0.0.1:5000` 또는 `http://localhost:5000`으로 접속합니다.

### 5.3. 게임 플레이

1.  웹 페이지가 로드되면 "새 게임 시작" 버튼을 클릭하여 게임을 시작합니다.
2.  **기물 이동**:
    *   자신의 기물(한나라, 아래쪽)을 클릭합니다. 파란색 원이 표시됩니다.
    *   이동 가능한 위치에 녹색 사각형이 표시됩니다.
    *   녹색 사각형이 표시된 칸을 클릭하면 기물이 이동합니다.
    *   기물이 이동하면 효과음이 재생되고, AI의 턴으로 넘어갑니다.
3.  **AI의 턴**: AI가 수를 계산하고 자동으로 기물을 이동합니다. AI가 수를 두면 효과음이 재생되고, AI 평가(점수/메이트)가 UI에 표시됩니다.
4.  **디버깅**: 문제가 발생하면 웹 브라우저의 개발자 도구(F12)를 열어 'Console' 탭과 'Network' 탭을 확인하면 유용한 정보를 얻을 수 있습니다. Flask 서버의 터미널 출력도 확인하세요.

## 6. 향후 개선 사항

*   **게임 종료 조건**: 장군, 외통, 멍군(무승부) 등의 게임 종료 조건을 정확하게 판단하고 UI에 표시하는 로직 추가.
*   **기물 잡기 애니메이션**: 기물이 잡혔을 때 시각적인 효과 추가.
*   **이동 기록**: 게임의 모든 이동을 기록하고 표시하는 기능.
*   **되돌리기/다시하기**: 수 되돌리기/다시하기 기능 추가.
*   **다양한 게임 모드**: AI 난이도 조절, 2인 플레이어 모드 등.
*   **UI/UX 개선**: 드래그 앤 드롭 기물 이동, 더 부드러운 애니메이션, 반응형 디자인 등.
*   **오류 처리 강화**: 사용자 입력 오류, 엔진 통신 오류 등에 대한 더 견고한 처리.
